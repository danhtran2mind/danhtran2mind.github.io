
# DanhTran2Mind Official Website

> **Info:** This repository is the DanhTran2Mind official website built using the `al-folio` Jekyll theme.

- **Tags:** infor, note, information

This repository contains a local copy of the al-folio Jekyll theme used to build academic websites.

## Summary

- A simple, clean, and responsive Jekyll theme for academics.
- This repository includes documentation and customization files in the `docs/` and `_includes/` directories.
